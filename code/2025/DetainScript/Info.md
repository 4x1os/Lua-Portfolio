A former detain tool that uses a weld cuff rather than using CFrame updates every tick to keep the user in place.

The detain tool features a highlight to indicate player to arrest, and provides an interface to the user to provide
a duration and reason of arrest. It also disables the inventory and prevents players from resetting

This is an older script. Newer changes added including making a server keep track of detained users, keeping their jailed status even if they left the server, and adding additional checks to the server in case someone was able to manipulate the client side into accessing the inventory or resetting.
